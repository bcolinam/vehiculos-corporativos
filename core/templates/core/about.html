Desarrollador Full Stack & Ingeniero en Sistemas

Especializado en desarrollo web con Python/Django, Data Analytics y arquitecturas de software escalables. Con experiencia en el mercado chileno y apasionado por crear soluciones tecnológicas eficientes.

Este sistema de gestión vehicular fue desarrollado como proyecto final del curso de Python con Django, demostrando habilidades en construcción de aplicaciones web profesionales, implementación de autenticación segura, gestión de permisos y creación de interfaces de usuario intuitivas.

El objetivo principal es proporcionar una solución completa para la administración de flotas vehiculares, con funcionalidades que incluyen registro de vehículos, asignación de responsables, control de estado y generación de métricas en tiempo real.

Tecnologías Implementadas

Django Framework

Backend robusto con ORM, autenticación y sistema de templates

Base de Datos

PostgreSQL/MySQL con relaciones complejas y migraciones

Seguridad

Autenticación, CSRF protection, permisos y roles de usuario

Frontend Moderno

HTML5, CSS3, JavaScript y Chart.js para visualizaciones

Características Principales

  • Sistema de autenticación completo con roles de usuario (admin/regular) y gestión de permisos granular

  • CRUD completo de vehículos con atributos detallados, estados y asignación a usuarios responsables

  • Dashboard interactivo con gráficos en tiempo real, métricas y filtros avanzados para análisis de datos

  • Arquitectura escalable diseñada para futuras ampliaciones como APIs REST, reporting avanzado e integraciones externas
