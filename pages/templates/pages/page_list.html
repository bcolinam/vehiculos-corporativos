Acerca de este proyecto

Mi nombre es Billy Colina y este proyecto fue desarrollado como entrega final del curso de Python con Django.

El objetivo principal es demostrar la construcción de una aplicación web profesional utilizando:

  • Django Framework
  • Autenticación y roles de usuario
  • Templates con herencia
  • Vistas basadas en funciones y clases
  • Manejo de formularios y permisos
  • Arquitectura limpia y escalable

El sistema fue diseñado con un enfoque modular para facilitar futuras ampliaciones como reporting avanzado, APIs o integraciones externas.
